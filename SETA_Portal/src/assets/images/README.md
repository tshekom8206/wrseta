# Images

This folder contains static image assets for the application.

## Usage

Images placed in this folder can be referenced in your components using the `/assets/images/` path:

```typescript
// In component template
<img src="/assets/images/logo.png" alt="Logo" />

// Or in component TypeScript
imageUrl = '/assets/images/logo.png';
```

## Structure

Organize images into subfolders as needed:
- `/logos/` - Logo files
- `/icons/` - Icon images
- `/backgrounds/` - Background images
- `/profile/` - User profile images
- `/ui/` - UI element images
